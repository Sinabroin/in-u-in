# IN U IN - 정성적 인터뷰 매칭 플랫폼

정성적 인터뷰 매칭을 보다 쉽고 간편하게 제공하는 플랫폼입니다.

## 주요 기능

- 정성적 인터뷰 참여자 모집
- 인터뷰 제안 기능
- 참여할 인터뷰 탐색
- 맞춤형 인터뷰 매칭
- 인터뷰 관리 및 분석

## 기술 스택

- Next.js 14
- TypeScript
- Tailwind CSS
- Prisma
- Google Sheets API

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 프로덕션 서버 실행
npm start
